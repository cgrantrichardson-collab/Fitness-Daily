
# Fitness Daily (PWA)
A tiny offline-capable web app that shows your **daily** training plan and **nutrition checklist**.

## Install on iPhone
1. Unzip and open `index.html` in Safari (or host these files).
2. Tap Share → **Add to Home Screen**.
3. Open from your home screen.

## Files
- `index.html` – the app (UI & logic)
- `manifest.webmanifest` – Add-to-Home-Screen metadata
- `sw.js` – offline caching
- Start date defaults to **2025-09-01**; change it in the app.
